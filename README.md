# WEBGAMEUI Portfolio Website

## Deskripsi
Proyek ini adalah website portofolio web yang menampilkan semua program dan proyek yang dibuat untuk menunjukkan kemampuan dalam pengembangan web. Website ini dirancang untuk menampilkan halaman depan, proyek, tentang saya, kontak, dan demo setiap aplikasi web.
Contoh Tampilan:
muhammadfariqzain.it.student.pens.ac.id

## Isi Portofolio
- Halaman beranda dengan ringkasan kemampuan dan layanan
- Daftar proyek web yang telah dibuat
- Demo aplikasi web interaktif
- Informasi tentang diri dan pengalaman
- Formulir kontak untuk klien dan kolaborator

## Teknologi
- HTML
- CSS
- JavaScript
- (Opsional) Framework ringan atau library frontend jika digunakan

## Cara Menjalankan
1. Buka folder `WEBGAMEUI`.
2. Jalankan file `index.html` di browser.
3. Untuk pengembangan lanjutan, gunakan server lokal seperti Live Server di VS Code.

## Struktur Proyek
- `index.html` - halaman utama portofolio
- `style.css` - gaya tampilan dan tata letak
- `script.js` - interaksi dan logika JavaScript
- `assets/` - gambar, ikon, atau sumber daya pendukung

## Fitur Utama
- Navigasi antar halaman yang responsif
- Tampilan proyek yang interaktif
- Demo aplikasi web yang mudah diakses
- Desain modern dan profesional

## Kontak
Jika ingin melihat lebih banyak proyek atau bekerja sama, silakan hubungi melalui formulir kontak di halaman portofolio.
